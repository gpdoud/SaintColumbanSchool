# Thinkpad Setup

## ISSUE

Setting up the Thinkpad laptops for student use.

## RESOLUTION

1. Connect wifi to StCS-Staff
2. Run Windows Update to get all updates installed
3. Check Advanced Options > Optional Updates and apply if exists
4. Install Google Chrome
5. Install Microsoft365 and activate all apps (requires an Microsoft admin email and password)
    - go to url: `office.com`
    - sign-in with Microsoft credentials
    - select `apps`
    - select `install apps`
6. Type "Troubleshoot Windows Update" and run the troubleshooter
7. Check Setup > System > About
    - Check that Windwows Specification shows '25H2'
